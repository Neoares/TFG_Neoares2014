Download this project as a .zip file Download this project as a tar.gz file

Resumen

Este proyecto consiste en la elaboración de un juego de navegador en tiempo real con Node.js. Tanto el cliente y el servidor se han programado en JavaScript, y se ha utilizado una base de datos NoSQL llamada MongoDB. También se han añadido WebSockets, de manera que establecemos nuestro propio protocolo de comunicación entre el cliente y el servidor, dejando de lado el típico protocolo HTTP.
