Resumen

Español:
Este proyecto consiste en la elaboración de un juego de navegador en tiempo real con Node.js. Tanto el cliente como el servidor se han programado en JavaScript, y se ha utilizado una base de datos NoSQL llamada MongoDB. También se han añadido WebSockets, de manera que establecemos nuestro propio protocolo de comunicación entre el cliente y el servidor, dejando de lado el típico protocolo HTTP.

Català:
Aquest projecte consisteix en la elaboració d’un joc de navegador en temps real amb Node.js. Tant el client com el servidor han estat programats en JavaScript, i s’ha utilitzat una base de dades NoSQL, MongoDB. També s’han afegit WebSockets, de manera que establim el nostre propi protocol de comunicació entre el client i el servidor, deixant de banda el típic protocol HTTP.

English:
This project involves developing a Realtime browser game with Node.js. Both client and server are programmed in JavaScript, and we’ve used a NoSQL database called MongoDB. We’ve also added WebSockets, so we can set our own communication protocol between client and server, instead of the typical HTTP protocol.
